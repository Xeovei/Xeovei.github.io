T422584 mirror

2024-02-01 1 min read # luogu

题意

给你一个字符串，请你输出它镜面翻转的样子

样例

输入
01258AHIJLMOTUVWXY
输出
YXWVUTOMJLIHA82510
样例解释:第一个样例包含所有可能出现的字符

思路

1.将整个字符串倒序
2.因为是镜面,而样例中只有25JL需要翻转，所以只需要特判一下就可以了

code↓

#include <bits/stdc++.h>
using namespace std;
string s;
int main(){
//	freopen("mirror.in","r",stdin);
//	freopen("mirror.out","w",stdout);
	cin>>s;
	int n=s.length();
	for(int i=n-1;i>=0;i--){
		if(s[i]=='J') cout<<"L";
		else if(s[i]=='L') cout<<"J";
		else if(s[i]=='2') cout<<"5";
		else if(s[i]=='5') cout<<"2";
		else cout<<s[i];
	}
	return 0;
}
